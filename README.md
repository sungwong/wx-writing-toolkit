# wx-writing-toolkit

公众号写作工具箱 skill，适用于 Claude Code。

填入你的领域和目标人群，直接生成可用内容——不用自己想提示词，不用反复调整，开箱即用。

---

## 这个 skill 有什么用？

做公众号最耗时的不是写，是**想不出来写什么、不知道怎么开头、写完不知道结尾怎么收**。

这个工具箱把公众号写作拆成 7 大环节，每个环节都有专门的提示词工具，你只需要告诉它你的领域和读者，它帮你把剩下的做掉。

**你填入：** 领域 + 目标人群  
**它输出：** 可直接用的内容

---

## 安装

```bash
npx skills add sungwong/wx-writing-toolkit
```

## 配置（一次性，5分钟）

安装后，创建配置文件 `~/.wx-writing-toolkit/config.md`：

```markdown
# wx-writing-toolkit 配置

领域: 你的公众号方向（例：职场效率、亲子教育、健身减脂）
人群: 你的目标读者（例：25-35岁职场妈妈、想副业的上班族）
账号名: 你的公众号名称（可选）
```

配置一次，之后每次用都自动读取。

## 使用

输入 `/wx-writing-toolkit` 打开工具菜单，选择想用的工具编号即可。

或者直接说需求：
- 「帮我做用户画像」→ 自动调用 1.1
- 「帮我想10个选题」→ 自动调用 2.1
- 「帮我写开头」→ 询问你想用哪种开头类型
- 「把这段话润色一下」→ 自动调用 5.2
- 「帮我写结尾」→ 询问你想要哪种风格
