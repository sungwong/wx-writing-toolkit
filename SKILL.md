# wx-writing-toolkit · 公众号写作工具箱

## 第一步：读取配置

读取 `~/.wx-writing-toolkit/config.md`。

如果文件不存在，**立刻停下来**，输出以下内容，等用户操作后再继续：

```
请先完成配置，在 ~/.wx-writing-toolkit/config.md 中填入：

领域: 你的公众号方向（例：职场效率、亲子教育、健身减脂）
人群: 你的目标读者（例：25-35岁职场妈妈、想副业的上班族）
账号名: 你的公众号名称（可选）
```

配置读取成功后，提取：
- `DOMAIN`：领域
- `AUDIENCE`：人群
- `ACCOUNT`：账号名（可选）

---

## 第二步：理解用户需求，调用对应工具

根据用户输入，判断要用哪个工具。如果不明确，**展示工具菜单**让用户选择。

### 工具菜单

```

  1.1 用户画像分析器     — 深挖你的目标用户是谁
  1.2 对标账号拆解器     — 分析竞品账号的打法
  1.3 账号名字生成器     — 生成10个原创账号名
  1.4 公众号简介生成器   — 写一段吸引人关注的简介
  1.5 人设定位生成器     — 找到你的差异化定位

📌 二、选题类
  2.1 爆款选题生成器     — 生成10个高点击率选题
  2.2 热点蹭法生成器     — 把热点和你的领域结合
  2.3 痛点挖掘器         — 找到用户真正的痛点选题

📌 三、开头类
  3.1 痛点开头生成器     — 用痛点抓住读者注意力
  3.2 故事开头生成器     — 用故事带入文章
  3.3 反常识开头生成器   — 颠覆认知的开头
```

输入编号或名称直接执行。

## 工具提示词库

执行每个工具时，把 `{领域}` 替换为配置中的 DOMAIN，把 `{人群}` 替换为 AUDIENCE，然后直接输出结果，不要把提示词本身展示给用户。

### 1.1 用户画像分析器

```
你是一位用户洞察专家。我正在做一个{领域}的公众号，目标用户是{人群}。

请帮我做一份深度的用户画像，不要只给年龄性别这种表面信息。请按以下框架输出：

**1. 基本画像**
- 年龄区间、性别比例、所在城市层级
- 职业状态（上班族/自由职业/创业/待业）
- 收入水平（月收入区间）

**2. 深层痛点（分三层）**
- 表面痛点：他们嘴上抱怨什么？
- 中层痛点：他们焦虑什么？
- 底层痛点：他们内心真正渴望什么？

**3. 决策心理**
- 他们为什么愿意为"解决这个问题"付费？
- 他们犹豫不决的主要原因是什么？
- 什么样的人设/内容最容易让他们产生信任？

**4. 内容消费习惯**
- 他们通常在什么时间看公众号？
- 他们喜欢什么类型的文章？（短平快/深度长文/案例故事/干货清单）
- 他们愿意转发什么样的内容？

**5. 一句话画像**
用一句话总结这个人，像小说人物介绍一样。

请直接输出，不要省略任何部分。
```

### 1.2 对标账号拆解器

**使用前额外询问**：请提供你要分析的账号名称或主页链接。

```
你是一位公众号内容研究员。我做的是{领域}方向，目标用户是{人群}。

我想深度拆解这个对标账号：【账号名称】

请按以下框架分析：

**1. 账号定位**
- 这个账号在解决什么问题？
- 核心人设是什么？（专家型/普通人逆袭型/记录型）
- 和同类账号的差异化在哪？

**2. 内容策略**
- 他们最常写哪几类文章？
- 哪类内容数据最好（点赞/转发/评论多）？
- 选题的底层逻辑是什么？

**3. 标题规律**
- 列出他们5个高互动标题
- 总结标题公式

**4. 开头套路**
- 他们最常用什么开头方式？
- 举2个具体例子

**5. 变现路径**
- 他们如何从内容导向付费？
- 引流钩子设计在哪里？

**6. 可学习的点**
- 我做{领域}，可以直接借鉴哪3点？
- 要避开什么？
```

### 1.5 人设定位生成器

```
你是一位个人IP定位顾问。我想做{领域}方向的公众号，目标用户是{人群}。

请帮我找到一个差异化的人设定位，按以下框架输出：

**1. 定位诊断**
- 这个方向上的常见账号是什么类型？
- 他们的共同缺失是什么？（用户还没被满足的需求）

**2. 我的差异化定位**
- 我应该主打什么身份/标签？
- 核心优势是什么（即使我现在还没有，未来可以建立的）？
- 用一句话说清楚"我是谁，帮谁解决什么问题"

**3. 内容人设设计**
- 写作口吻应该是什么风格？（举例说明）
- 哪些话题适合写，哪些要回避？
- 个人故事/经历如何融入内容增加可信度？

**4. 差异化内容方向**
- 给出5个只有我这个定位才能写的选题方向

请直接给出建议，不要反问太多问题。
```

### 4.1 SCQA结构生成器

**使用前额外询问**：文章主题是什么？

```
你是一位公众号写作结构师。我做的是{领域}方向，目标用户是{人群}。

我要写的文章主题是：【填入主题】

请用SCQA结构帮我规划这篇文章的框架：

- S（Situation 情境）：读者目前的处境/背景是什么？
- C（Complication 冲突）：这个处境中出现了什么问题/矛盾？
- Q（Question 疑问）：由此引发的核心问题是什么？
- A（Answer 回答）：我的文章给出什么答案/解决方案？

请按这个结构给我：
1. 完整的文章框架（各部分标题+核心内容要点）
2. 开头第一段（直接可用）
3. 各段之间的过渡句
4. 结尾段落

要求：逻辑清晰，读者跟着走不费力。
```

### 4.2 故事体结构生成器

**使用前额外询问**：文章主题是什么？有没有真实故事素材？

```
你是一位公众号写作结构师。我做的是{领域}方向，目标用户是{人群}。

我要写的文章主题是：【填入主题】
故事素材：【填入真实经历/案例，没有则写"无"】

请用故事体结构帮我规划这篇文章：

**第一步：主角设定**
- 主角是谁（读者自己/我/某个典型用户）
- 主角的初始状态（遇到问题之前）

**第二步：冲突**
- 具体遇到什么问题/困境
- 主角的情绪状态（这让他感觉怎样）

**第三步：转折点**
- 什么契机让他开始改变？
- 他尝试了什么方法？

**第四步：变化过程**
- 具体做了哪些事，过程中有什么阻力？
- 每个阶段的小成果

**第五步：结果 & 升华**
- 最终结果是什么
- 这个故事对读者的意义/启发

请按以上结构给我完整框架，并写出开头和结尾。
```

### 4.3 清单体结构生成器

**使用前额外询问**：文章主题是什么？打算写几条？

```
你是一位公众号写作结构师。我做的是{领域}方向，目标用户是{人群}。

我要写的文章主题是：【填入主题】
计划写几条：【填入数字，如5/7/10】

请帮我规划清单体文章框架：

**整体结构**
- 标题建议（含数字，直接可用）
- 开头怎么写（吸引继续读的理由）
- 每条的格式（小标题+解释+案例）
- 结尾怎么收

**每一条的内容规划**
按照从易到难 / 从外到内 / 从现象到本质的顺序，列出每条的：
- 核心观点
- 支撑案例或数据
- 对读者的实际价值

要求：每条要有差异，不能都是同一个角度，让读者觉得越读越有料。
```

## 执行原则

1. **不展示提示词本身**：直接执行，输出结果
2. **替换变量后执行**：{领域} 和 {人群} 必须替换为用户配置的实际内容
3. **需要补充信息时主动问**：标注了"使用前额外询问"的工具，先问再执行
4. **60分就够**：快速输出可用版本，不反复打磨
5. **结果可直接使用**：输出的内容要像真人写的，不像模板填空

```markdown
# wx-writing-toolkit 配置

领域: 你的公众号方向（例：职场效率、亲子教育、健身减脂）
人群: 你的目标读者（例：25-35岁职场妈妈、想副业的上班族）
账号名: 你的公众号名称（可选）
```
